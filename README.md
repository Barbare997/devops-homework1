# DevOps Homework 1 - Todo App

A simple Todo web application built with Node.js + Express to demonstrate CI/CD pipeline automation and deployment strategies.

## Tech Stack

- Node.js
- Express
- Jest + Supertest
- HTML/CSS/Vanilla JavaScript

## Project Structure

```
.
├── public/
│   ├── app.js
│   ├── index.html
│   └── styles.css
├── src/
│   ├── app.js
│   └── server.js
├── tests/
│   └── app.test.js
└── package.json
```

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm start
   ```
3. Open:
   ```
   http://localhost:3000
   ```

## Run Tests

```bash
npm test
```
